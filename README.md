# PHP FizzBuzz Kata

This Kata is inspired by [Coding Dojo's FizzBuzz Kata](http://codingdojo.org/kata/FizzBuzzi/).


## Requirements

* PHP >= 7.0.0
* Composer


## Install & Run tests

After cloning this repository, run the following commands:

```
$ composer install
$ ./vendor/bin/phpunit tests --colors=always
```
